PROYECTOS

Gimnasio CPFSJ

Django App

La aplicación permite llevar un seguimiento de las rutinas de entrenamiento de cada uno de los alumnos del gimnasio. Los entrenadores pueden supervisar y ajustar las rutinas de los usuarios en función de su progreso, utilizando el panel de administración. Estos cambios se reflejan en una base de datos en Postgres."

Por su parte, los alumnos siguen las instrucciones de su entrenador a través de la aplicación. Dado que la aplicación es responsive, los usuarios pueden acceder a sus rutinas desde sus dispositivos móviles en el momento de entrenar.

Pizza World

JavaScript Vanilla

Esta es una página de portafolio desarrollada sin el uso de frameworks ni librerías externas, utilizando únicamente HTML, CSS y JavaScript. La web es completamente responsive, lo que significa que se adapta a cualquier dispositivo.

Está enfocada en la experiencia del usuario en el front-end, con efectos implementados en JavaScript y estilos elaborados en CSS.

Tecno Market

Django App

TecnoMarket es una tienda en línea especializada en productos electrónicos, creada con el framework Django. Además, se han incorporado conceptos de JavaScript para darle un toque de dinamismo.

Una de las ventajas de esta aplicación es su eficiente panel de administración, el cual facilita el control y la gestión de productos. Esto hace que el mantenimiento de la tienda sea una tarea amigable y accesible.

Task Mngr

React - Django Rest Framework

Task mngr es una aplicación que te permite la creación de listas de tareas para una mejor organización.

La interfaz de usuario está desarrollada con React. En cuanto al backend, Task Mngr hace uso de Django Rest Framework para exponer una API que es consumida por el front-end, permitiendo un manejo completo de las operaciones CRUD (Crear, Leer, Actualizar y Eliminar) de tareas.
